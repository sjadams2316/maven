## Advisor Dashboard (MAJOR BUILD)
Built complete advisor control plane at `/advisor`:

### Pages Created:
1. **`/advisor`** - Advisor home
   - Stats: Total AUM, active clients, alerts
   - Upcoming meetings with quick prep links
   - Insights requiring attention
   - Recent client activity feed
   - Quick actions

2. **`/advisor/clients`** - Client list
   - Search, filter (all/needs attention/inactive), sort
   - Shows AUM, last login, alerts, tone setting
   - Click-through to detail

3. **`/advisor/clients/[id]`** - Client detail
   - 5 tabs: Portfolio, Insights, Activity, Notes, Settings
   - **Portfolio**: Allocation chart, accounts, holdings table with gains
   - **Insights**: Per-insight curation controls (visibility dropdown + advisor notes)
   - **Activity**: Login/page view history
   - **Notes**: Free-form advisor notes
   - **Settings**: Client tone (conservative/moderate/engaged)

4. **`/advisor/clients/[id]/prep`** - Meeting prep
   - Portfolio summary with changes since last meeting
   - Auto-generated talking points by priority
   - Action items checklist (with completion toggle)
   - Suggested agenda with timing
   - Generate PDF / Email to client buttons

5. **`/advisor/insights`** - Global insights view
   - All insights across all clients
   - Filter by severity, type, search
   - Shows potential savings rollup

### Key Features:
- Insight curation: show / show_with_context / advisor_only / discussion
- Client tone settings control Maven's proactive behavior
- Advisor notes on insights persist
- Meeting prep generates talking points from insights
